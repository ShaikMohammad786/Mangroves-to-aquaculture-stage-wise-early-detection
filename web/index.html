📖 Methodology

How the detection system works

🧠 Core Innovation: Biological Pathway Constraint

Unlike traditional change detection that compares two dates, this system tracks the biological transition pathway that mangrove-to-aquaculture conversion follows:

S1
Dense Mangrove
→
S2
Degradation
→
S3
Clearing
→
S4
Water Filling
→
S5
Operational Pond

Operational mode enforces sequential transitions. Historical mode allows jump transitions because multi-year epochs can miss intermediate observations.

🛰 Multi-Sensor Evidence

EvidenceWeightSource
Optical30%NDVI, MNDWI, EVI from Landsat/S2
SAR30%VV, VH backscatter from Sentinel-1
Shape20%Rectangularity, edge density
Persistence20%JRC water occurrence history

Confidence threshold: ≥ 85% for confirmed detection.

⚙ Processing Pipeline

1. AOI Definition & DEM mask
2. Satellite data acquisition (GEE)
3. Cloud masking & radiometric calibration
4. Z-score normalization per band
5. Speckle filtering (Refined Lee) for SAR
6. Feature extraction (vegetation + water + SAR + shape)
7. Stage classification with pathway constraints
8. Multi-evidence validation with FP suppression
9. Polygon extraction & alert generation
10. Accuracy validation (GMW + JRC comparison)

🔬 Key Formulas

NDVI (NIR − RED) / (NIR + RED)
MNDWI (GREEN − SWIR1) / (GREEN + SWIR1)
AWEI 4(GREEN − SWIR) − (0.25·NIR + 2.75·SWIR2)
Speckle Filter x̂ = x̄ + W(x − x̄), W = (σ² − n²)/σ²
Z-Score I' = (I − μ) / σ
Confidence 0.3·Opt + 0.3·SAR + 0.2·Shape + 0.2·Persist
